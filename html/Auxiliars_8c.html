int  nTest,

)

This function is designed to insert a test in the first free position in the vector. The main diference with insertResult is related with the order. In this case we can have a test vector such [t2,t3,t5] since with insertResult we'd have [null,null,t2,t3,null,t5]. We have designed it to achieve a better performance

Parameters
nMutantMutant where the test will be inserted
pTestTest to insert
Returns
0 if the mutant is not allocated
